Compare Size Effect of Spatial Units(SESU)

gdverse provides the sesu_opgd() and sesu_gozh() function to support the selection of optimal spatial analysis scales which based on OPGD and GOZH respectively. Please refer to the help documentation of the corresponding function for more details.

Here, we use FVC raster data as an example to demonstrate the optimal spatial analysis scale selection function in gdverse.

Processing data

First, we construct FVC data under different spatial units using the original data.

library(terra)
library(tidyverse)
library(gdverse)
fvcpath = "https://github.com/SpatLyu/rdevdata/raw/main/FVC.tif"
fvc = terra::rast(paste0("/vsicurl/",fvcpath))

su = seq(1000,10000,by = 1000)

  purrr::map(\(.x) terra::aggregate(fvc,fact=.x ,fun="mean") %>%

               tibble::as_tibble())
fvc = c(list(fvc1000),fvc_other)
